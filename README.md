## Package Structure 📦

- `디자인패턴명` 패키지
    - 디자인 패턴에 맞는 README 설명
    - 디자인 패턴에 맞는 예제 프로그램 코드
    - `execute` 패키지
        - 예제 프로그램을 실행할 수 있는 Main 클래스 (main 함수 포함)
    - `practice` 패키지
        - 디자인 패턴 연습문제 README
        - (+) 필요할 시, 연습문제 소스 추가
